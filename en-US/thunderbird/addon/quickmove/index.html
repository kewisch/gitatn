About this extension

Quick Folder Move helps Thunderbird users file messages without leaving the keyboard. Search for a destination folder, move or copy the selected messages, jump to folders, and apply tags from the same compact popup.

It is designed for people with many folders and repeat filing workflows. The extension keeps recent destinations close at hand while still letting you search across the full folder tree.

  • Move, copy, go to folder, or tag selected messages.
  • Use shortcuts and a toolbar popup for fast filing.
  • Tune recent folders, excluded folders, and folder matching behavior.

Permissions and data

Required permissions:

  • Store data on your device
  • Read and modify message compose windows
  • Read your mail accounts and folders
  • Read email messages
  • Move email messages
  • Apply and remove message tags
  • Read your message tags
  • Update email message properties

Data collection:

The developer says this extension does not require data collection.
